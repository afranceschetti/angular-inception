AngularJS Components Directives

Markers on DOM element tell AngularJS to attach a behavior to it or even to transform it

Different ways of use

<my-dir></my-dir>
by name
<span my-dir="exp"></span>
by attributes
<!-- directive: my-dir exp -->
by comments
<span class="my-dir: exp;"></span>
by class names

Directives ready to use

ng-change ng-click ng-disabled ng-if ng-include ng-model ng-repeat ng-show
